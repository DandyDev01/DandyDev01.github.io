Inventory Manager

This is a project I created for one of my classes, Human Computer Interaction. The purpose of this project was to apply what we learned during the course to the design of an application. For me, this meant a simple design that would be easy for the user to navigate, find what they needed to complete tasks, allow for flexibility, and agility.

Another part of the system that was very important was data validation and making sure the user didn't accidently enter in data values that would cause errors later in the program.
