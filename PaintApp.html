Paint App

This is a small painting application made using the Processing programming language. I created this little program because I've wanted to make a painting app for quite a while and finally had a chance. I chose to use Processing because it gives easy access to drawing graphics to the screen, which was a big win given I wanted to create my own UI elements from scratch. This included buttons, sliders, and text fields.

Currently the program only has three brushes, but I want to add more and add functionallity for users to create their own brushes.
