Learn Unity

This project was for a course I took in web development. We were asked to choose a topic and create a website about it. I chose to make a website teaching people how to make 2D games in Unity.

Some of the specific site requirments were:

  1. Fully responsive, this meant that it needed to work on all devices regardless of screen size.
  2. Needed to work in all the major browsers. This included Firefox, Chrome, Edge, Samsung Internet, and Safari
  3. Needed to make use of some external API, I chose to use Youtubes.
  4. Use of some third party library, I chose react.
  5. Use of javascript, I implemented a quizing system, etc.
